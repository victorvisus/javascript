# Curso de JavaScript Moderno

Este workspace contiene el material y proyectos para el curso de JavaScript Moderno. Aprenderás a consumir distintas REST APIs con JSON y Fetch API + Async/Await, uno de los temas más importantes como desarrollador JavaScript.

## Descripción del Curso

JavaScript es el lenguaje más buscado por los empleadores hoy en día. En este curso aprenderás:

- Lo básico del lenguaje así como conceptos más avanzados
- Programación orientada a objetos con Classes (ES6) y Prototypes (ES5)
- Conceptos nuevos en ES6 como Template Literals, arrow functions, generadores, iteradores, promises, async/await, fetch api y mucho más
- Programación Asíncrona con Promises, Callbacks y Async Await
- Construcción de una gran cantidad de proyectos, consumiendo REST API's y otros con código puro de JavaScript
- Agregar interacción a tus aplicaciones y páginas web

## Cómo Usar

1. Navega a la carpeta del tema que deseas estudiar.
2. Abre el archivo `index.html` en tu navegador para ver la lección o proyecto.
3. Revisa los archivos JS para entender el código.
4. Modifica y experimenta con el código para aprender.

## Requisitos

- Un navegador web moderno
- Un editor de código (recomendado: VS Code)
- Conocimientos básicos de HTML y CSS (útil pero no obligatorio)
